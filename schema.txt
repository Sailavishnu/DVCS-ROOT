1. users
   Stores registered user accounts.
────────────────────────────────────────────────────────────────────────────────
  Attribute           | Type          | Description
  ─────────────────── | ───────────── | ─────────────────────────────────────────
  _id                 | ObjectId (PK) | Unique user identifier
  username            | String        | Unique login username
  displayName         | String        | Flat full name for display (NEW)
  name                | Document      | Compound: { firstName, lastName }
  passwordHash        | String        | BCrypt password hash
  role                | String        | 'user' or 'collaborator'

  filesLimit          | Integer       | Max number of files allowed
  lastActiveAt        | Date          | Timestamp of last activity
  collabStatus        | String        | 'active' or 'inactive'
  totalFilesShared    | Integer       | Derived: count of files shared
  createdAt           | Date          | Account creation timestamp

  Attribute           | Type          | Description
  ─────────────────── | ───────────── | ─────────────────────────────────────────
  _id                 | ObjectId (PK) | Unique file identifier

5. file_snapshots

────────────────────────────────────────────────────────────────────────────────
  Attribute           | Type          | Description
  ─────────────────── | ───────────── | ─────────────────────────────────────────
